# Bichomanía – Clothing Store (Frontend)

Frontend de una tienda de ropa desarrollado con **React**.

## Tecnologías
- React
- JavaScript
- HTML
- CSS
- npm

## Estructura del proyecto
/src
/public
package.json
## Cómo ejecutar el proyecto
1. Clonar el repositorio:
```bash
git clone https://github.com/RodrigoAndre1321/Clothing-Store-Bichomania-frontend.git
```
2. Instalar dependencias:
npm install
3. Ejecutar el proyecto:
npm start
## Backend: El backend está desplegado en Clever Cloud (repositorio separado).
